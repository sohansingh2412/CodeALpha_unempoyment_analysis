# 📉 Unemployment Analysis with Python

This project involves analyzing the unemployment rate across various regions. Using data visualization and statistical analysis, I explored the trends and impacts of unemployment, especially focusing on periods like the COVID-19 pandemic.

## 📋 Project Overview
- **Objective:** To visualize and analyze the unemployment rate to understand geographic and temporal trends.
- **Dataset:** Contains regional data including Estimated Unemployment Rate (%), Estimated Employed, and Estimated Labour Participation Rate.
- **Key Focus:** Analyzing how different states performed during economic shifts.

## 🛠️ Tech Stack
* **Language:** Python 🐍
* **Libraries:** * `Pandas` (Data Processing)
  * `Matplotlib` & `Seaborn` (Static Visualizations)
  * `Plotly` (Interactive Charts)

## 🚀 Key Steps
1. **Data Cleaning:** Handling missing values and ensuring correct date-time formatting.
2. **EDA:** Creating bar charts, line graphs, and sunburst plots to compare regions.
3. **Insights:** Identifying the months and regions with the highest peak in unemployment.
